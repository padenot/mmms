# `mmms`

**M**onome **M**onophonic **M**elody **M**equencer.

Code intended to run on a [bela](https://bela.io), to sequence pitched
instruments, triggered using gates. Sequencing is done on a [monome
grid](https://monome.org/docs/grid/).
